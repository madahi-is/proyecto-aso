# Proyecto ASO

Proyecto Python que utiliza **Pillow** para manejo de imágenes y **Tkinter** para interfaces gráficas.

## Requisitos

- Python 3.6 o superior
- pip (gestor de paquetes de Python)
- zypper (gestor de paquetes de OpenSUSE)

## Instalación en OpenSUSE

1. **Instalar dependencias del sistema**

sudo zypper install python3-tk python3-Pillow-tk

2. **Clonar el repositorio**
 
git clone https://github.com/madahi-is/proyecto-aso.git
cd proyecto-aso
